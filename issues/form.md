### Deskripsi
Optimalkan proses input tanggal pada form input SEP manual agar lebih cepat dan minim human error.

### Perubahan yang Dibutuhkan
Tambahkan field baru berbentuk dropdown:
- Bulan
- Tahun

### Behavior
- Secara default, field otomatis terisi menggunakan bulan dan tahun saat ini.
- User tetap dapat mengubah nilai dropdown jika diperlukan.

### Tujuan
- Mempercepat proses input SEP manual.
- Mengurangi kesalahan input tanggal.
- Meningkatkan konsistensi data.
